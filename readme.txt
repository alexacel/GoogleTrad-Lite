This is my translation web service
.

This project has been developed on Windows 10

######### Getting started #########
1-install node js from http://nodejs.org
2-install mongodb from http://www.mongodb.com
3-Open a node js console from the root directory of the project and run :
	> node install
4-Launch the script "launchMongoDb"
5-From the node js console run :
	> node server.js
6-Open a web browser and go to http://localhost:8080
7-Enjoy !


######### Running tests #########
1-The web server must be running
2-use another node js console and run
	>npm test
